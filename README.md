# bookmark2md

    Convert chrome bookmarks to md files and push them to GitHub repository.


## Use script
```bash
  // step 1
  cd Script
  // step 2
  export chrome bookmarks to `/Script/bookmarks.html`
  // step 3
  node bookmark2md.js
```
